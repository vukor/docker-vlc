docker-vlc
===========

This is a docker project for build and run vlc video translation and recording


How it's work
===========

1. Download project:

    ``$ git clone https://vukor@github.com/vukor/docker-vlc.git ~/docker``

2. Install docker on your system
3. In process..


share dirs
===========

``etc - configs files``

``rec - recording files``

``logs - vlc logs``
